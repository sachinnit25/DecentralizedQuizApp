# DecentralizedQuizApp
🌌 DecentralizedQuizApp

A decentralized quiz application built on the Stellar network that enables users to participate in quizzes, verify answers on-chain, and earn rewards in a transparent and trustless environment.

📌 Project Description

DecentralizedQuizApp is a blockchain-based application designed to bring fairness and transparency to online quizzes. Unlike traditional quiz systems, this app removes centralized control by using smart contracts powered by Soroban.

All quiz interactions — including answer validation and reward distribution — are handled securely on-chain, ensuring that results cannot be manipulated.

⚡ What It Does

Allows users to connect their wallet and participate in quizzes

Stores quiz data securely (on-chain or hybrid)

Verifies answers using smart contracts

Automatically rewards correct participants

Maintains transparent and immutable records of all attempts

🚀 Features

🧠 Decentralized quiz participation

🔐 Trustless answer verification

💸 Automated reward distribution

👛 Wallet-based authentication (Stellar accounts)

🌍 Transparent and tamper-proof system

⚡ Fast, low-cost transactions via Stellar

🧩 Built using Soroban smart contracts

🛠️ Tech Stack

Blockchain: Stellar

Smart Contracts: Soroban

Frontend: React / HTML / CSS

Backend: Node.js (optional)

Language: Rust (contracts), JavaScript
